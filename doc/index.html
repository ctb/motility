motility documentation

motility is a C++ library for searching DNA sequences with several different types of motif representations, from the simplest case (string literals!) to IUPAC-style motifs (e.g. WGATAR), Position-Weight Matrices (PWMs), and energy operators. It also has a simple Python interface which renders it almost immediately useful from the command line; the Python interface also supports reading of the JASPAR and TFD libraries of binding sites.

The current version of motility is 0.7, and it's available from the Cartwheel CVS archive on SourceForge under motility/, or upon request to titus@caltech.edu.
